This is a C# Console app Excercise/Project for an bank ATM.

The Goal for this project is to learn the basics of Classes,Methods,OOP,Synthax in C# and how to structure a complete console app.
The App should be able to recognise the user and the user can only acess his personal bank data if the PIN code mathces.

Inside of the app the user should be able to withdraw, deposit money.
Change his desired Balance between 3 Currencies and add a fourth one of his desire.
Change his PIN Code.

All the Data needs to be saved and loaded from a external JSON file.
